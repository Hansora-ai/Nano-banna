Instructions

  1. Choose Seedance 2 or Seedance 2 Lite.
  2. Add a prompt.
  3. Use either first/last frame mode or reference mode.
  4. Reference mode supports up to 9 images, 3 videos, and 3 audios.
  5. Reference video total and reference audio total must each stay within 15 seconds.
  6. Click Run. The result will be sent to your bot chat if credits are enough.

Important

Seedance 2 API does not allow first/last frames and multimodal references in the same request. This page shows both sections, but it will block that invalid combination before submit.
