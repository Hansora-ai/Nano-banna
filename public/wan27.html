Mode routing

  1. Prompt only or prompt + audio → Text to Video.
  2. First Frame, First + Last Frame, or video continuation → Image to Video.
  3. No Last Frame + reference images/videos → Reference to Video.
  4. Click Video Edit for video + reference images + prompt only.

Important

Video Edit allows only video, reference images, prompt, duration, ratio, and resolution. Its max duration is 10s. Other modes allow 4–15s.
